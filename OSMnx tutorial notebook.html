Introduction¶

This notebook will cover all basic elements of using OSMnx and Networkx for road network analysis, such as:

  • extracting basic statistics
  • understanding node and edge characteristics of road networks
  • identifying and visualizing shortest paths on folium

Plot Graph¶

OSMnx provides ways to query graphs from specific locations. graph_from_point allows us to specify a latitude and longitude (in that order) as a center point from which it returns the network within a certain distance.

You can specify what types of streets to extract by passing a string to network_type. These options include the following:

  • drive get drivable public streets (but not service roads)
  • drive_service get drivable public streets, including service roads
  • walk get all streets and paths that pedestrians can use (this network type ignores one-way directionality)
  • bike get all streets and paths that cyclists can use
  • all download all (non-private) OSM streets and paths
  • all_private download all OSM streets and paths, including private-access ones

The example below is a graph of the Legaspi Village area of Makati City.

In [8]:
lat = 14.556198
long = 121.017161

graph = osx.graph_from_point((lat, long), distance=1000, 
                             network_type='drive_service',
                             retain_all=True)
plot = osx.plot_graph(graph)

Setting the coordinate reference system¶

epsg: 32651 is a standard CRS (coordinate reference system) for most of Southeast Asia. Note that OSM is in epsg: 4326. We convert our crs only when computing distance values, which we will do later on.

In [9]:
graph = osx.project_graph(graph, to_crs={'init': 'epsg:32651'})

Plotting on Folium¶

Plotting on folium is slightly more complicated if you want to generate multiple independent paths. Below is code to generate paths that can be turned into geojson objects that can be plotted on both folium. Note that since path_edges is a dictionary, you can extract whichever features you want to use in the analysis later on. In this case, we're interested in the length of each edge.

In [30]:
path_edges = []
adj = graph.adj
for i in range(0,len(path)-1):
    edges = adj[path[i]][path[i+1]]
    min_length = float("inf")
    min_edge = None
    for key in edges: 
        edge = edges[key]
        if edge["length"] < min_length:
            min_length = edge["length"]
            min_edge = edge
    path_edges.append((path[i],path[i+1],min_edge))

def shortest_distance(graph, origin, o_c, dest, dest_c):
    """
    Creates a GeoDataFrame containing the edges of the shortest path
    from an origin to a destination point. These edges are represented 
    as linestrings that can be plotted on a map (folium works)
    
    Also prints the distance of the computed path.
    
    Parameters
    ----------
    graph : Graph 
        osmnx graph of the area containing the points
    
    origin : str
        name of origin point
        
    o_c : tuple of floats
        coordinates of origin point in Long/Lat format
    
    dest : str
        name of destination point
        
    dest_c : tuple of floats
        coordinates of destination point in Long/Lat format
        
    Returns
    -------
    path gdf : GeoDataFrame
    
    shorte
    """
    gdf = gpd.GeoDataFrame([{"name": origin, "geometry": Point(o_c)},
                            {"name": dest, "geometry": Point(dest_c)}], 
                           crs={'init':'epsg:4326'})
    gdf.to_crs({'init': 'epsg:32651'}, inplace=True)
    origin_point = gdf.loc[0].geometry
    dest_point = gdf.loc[1].geometry
    
    O = osx.get_nearest_node(graph, (origin_point.y, origin_point.x), 
                             method='euclidean', return_dist=True)
    D = osx.get_nearest_node(graph, (dest_point.y, dest_point.x), 
                             method='euclidean', return_dist=True)
    display(gdf)
    path = nx.shortest_path(graph, source=O[0], target=D[0])
    path_edges = []
    adj = graph.adj
    for i in range(0,len(path)-1):
        edges = adj[path[i]][path[i+1]]
        min_length = float("inf")
        min_edge = None
        for key in edges: 
            edge = edges[key]
            if edge["length"] < min_length:
                min_length = edge["length"]
                min_edge = edge
        path_edges.append((path[i],path[i+1],min_edge))
    node_dict = dict(graph.nodes)
    
    gdf_builder = [] 
    for edge in path_edges:
        u = node_dict[edge[0]]
        v = node_dict[edge[1]]
        curr = edge[2] 
        if "geometry" not in curr: 
            geom = LineString([(u['x'],u['y']),(v['x'],v['y'])])
        else:
            geom = curr['geometry']
        res = {"source": edge[0], "target": edge[1], 
           "distance": edge[2]['length'],"geometry":geom}
        gdf_builder.append(res)
    path_gdf = gpd.GeoDataFrame(gdf_builder,crs={'init':'epsg:32651'})
    path_gdf.to_crs({'init':'epsg:4326'},inplace=True)
    path_gdf
    
    shortest_dist = path_gdf['distance'].sum()
    print('distance:', shortest_dist)
    
    # compute shortest path
#     best_path = nx.shortest_path_length(graph, Point(o_c), Point(dest_c))   
    return path_gdf, shortest_dist
